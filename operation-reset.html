SOP: Heavy-Soil Dish Recovery

Restaurant mode

The secret is refusing to pay full price for scrub labor. Let heat + time do the ugly work.

Station Setup 5 min
  • Soak Station (bin or sink): hottest safe water + dish soap (+ optional baking soda).
  • Wash Station: fresh hot soapy water + brush/scrubber.
  • Dry Zone: rack/towel; air-dry preferred.
Workflow Soak first
  • Soak heavy items 30–120 min (or overnight).
  • Scrape after soak (spatula/scraper beats brute force).
  • Wash (then rinse).
  • Sanitize (hot rinse or approved kitchen sanitizer).
  • Dry (rack).
Rule: If it resists, it soaks again. Don’t escalate to misery.
Sorting Categories Speed
  • Light: cups/plates/utensils → dishwasher or quick wash.
  • Medium: bowls/saucepans → short soak + wash.
  • Heavy: baked-on cookware → long soak cycle.

Screenplay Log

Multi-day execution

Shot list vibes, but operational. Each beat ends in a measurable state: counters visible, dishes in flow, floors last.

EXT. WEATHERFORD SKY — MORNING OPEN

08:30

Mild daylight, cool edges. Windows crack. The plan begins with one sacrament: set the soak station before you touch anything else.

  • Soak bin established (hot water + soap).
  • All baked-on dishes go straight into soak.
  • Dish piles sorted by type (plates / cookware / utensils).

INT. MAIN KITCHEN — MIDDAY PIPELINE

11:30

Dishwasher becomes the engine. Clear one counter to become the clean staging surface.

  • Finish loading dishwasher → run.
  • Clear one full counter; nothing returns dirty.

INT. MESS HALL KITCHENETTE — AFTERNOON SCOPE

14:00

Kitchen-only scope, with one exception: the dining table is reclaimed as a temporary staging platform.

  • Counter sweep: everything off counters.
  • Dining table cleared enough to stage dishware + kitchen items.
  • Heavy dishes remain soaking; no scrubbing war.

INT. BOTH KITCHENS — CLOSE CLOSE

18:00

Easy dishes washed. Heavy items soak overnight. Dishwasher runs if half-full.

  • No dry dirty-dish piles left out.
  • Counters visible and wipeable zones established.

INT. BOTH KITCHENS — DISH MARATHON RESTORE

09:00–15:00

Dry, bright day: perfect for churn. Soak → scrape → wash cycles until the backlog collapses.

  • Empty dishwasher → reload → run (repeat as needed).
  • Scrub soaked cookware (scraper first).
  • Hand-wash overflow; put away immediately.
  • Once dishes are gone: wipe + sanitize all counters/sinks.

INT. BOTH KITCHENS — FLOORS BASE LAYER

09:30–12:30

Only after surfaces are clean do floors get their turn. Tile first, carpet last.

  • Main kitchen: sweep → mop → dry.
  • Mess hall kitchenette: vacuum slow passes → spot-treat stains.
Sequencing logic: Dishes → surfaces → floors. Anything else just smears the same problem around.

Permanent Close Procedure (10–15 min nightly)

Goal: never do a 3-day reset again.
Two rules:
  • Soak beats scrub — if it fights you, it soaks again.
  • Floors last — don’t clean what you’re about to re-dirty.
Optional Supplies List (minimal + cheap)
  • 2 plastic tubs/bins (for soak + transport)
  • Dish brush + scraper
  • Degreasing dish soap
  • Baking soda (optional but helpful)
  • Microfiber cloths / paper towels
  • Vacuum + spot cleaner for carpet day
